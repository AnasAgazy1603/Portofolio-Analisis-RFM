**Ringkasan Program**  

Program ini merupakan implementasi analisis **RFM (Recency, Frequency, Monetary)** untuk melakukan **segmentasi pelanggan** berdasarkan data transaksi penjualan sepeda. Proses analisis dilakukan menggunakan pendekatan **CRISP-DM (Cross-Industry Standard Process for Data Mining)** yang mencakup tahapan berikut:  

1. **Business Understanding** → Memahami tujuan bisnis, yaitu mengidentifikasi pola perilaku pelanggan untuk meningkatkan strategi pemasaran.  
2. **Data Understanding** → Mengeksplorasi dataset transaksi penjualan sepeda yang diperoleh dari asesor, termasuk pemeriksaan struktur data dan atribut penting.  
3. **Data Preparation** → Melakukan pembersihan data, menangani nilai yang hilang, dan menyiapkan data untuk analisis lebih lanjut.  
4. **Modeling**  
   - **Perhitungan RFM** → Menghitung nilai **Recency, Frequency, dan Monetary** untuk setiap pelanggan.  
   - **RFM Scoring** → Memberikan skor R, F, dan M berdasarkan kuartil data.  
   - **Segmentasi Pelanggan** → Mengklasifikasikan pelanggan ke dalam kategori **High-Value, Mid-Value, dan Low-Value Customers** berdasarkan skor RFM mereka.  
5. **Evaluation** → Menganalisis hasil segmentasi dengan visualisasi distribusi pelanggan dalam setiap kategori.  
6. **Deployment** → Menyusun strategi pemasaran yang disesuaikan untuk setiap segmen pelanggan guna meningkatkan loyalitas dan pendapatan bisnis.  

Dengan adanya analisis ini, bisnis dapat mengoptimalkan **strategi pemasaran berbasis data**, meningkatkan **customer retention**, serta memberikan **penawaran yang lebih personal** sesuai dengan pola pembelian pelanggan. Program ini dikembangkan menggunakan **Python** dengan berbagai teknik pemrosesan data dan analisis statistik.
